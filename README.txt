IronDiag phone app prototype

To use:
1. Unzip this folder.
2. Upload the folder to any simple web host, GitHub Pages, Netlify, or your own site.
3. Open index.html from your phone through the hosted link.
4. On iPhone: tap Share > Add to Home Screen.
5. On Android: tap browser menu > Add to Home screen or Install app.

Important:
- Saved jobs are stored locally on the device/browser.
- This prototype is not a replacement for the service manual.
- Always support raised equipment safely before testing hydraulics.
